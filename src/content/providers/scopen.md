## ERPNext Experience

### Since when have you been implementing ERPNext?

Since 2020 (founding); official Frappe partnership since 2023

### Do you contribute to ERPNext core or maintain publicly available Frappe apps? If yes, please link them.

Yes. Maintains and develops ERPNext France app and multiple related apps:
- ERPNext France (https://github.com/scopen-coop/erpnext_france) - French compliance features including FEC (Fichier des Écritures Comptables), transaction logs, VAT verification
- Job Card Planning (https://github.com/scopen-coop/jobcard_planning) - Calendar-based production planning
- Special Item Accountancy Code (https://github.com/scopen-coop/special_item_accountancy_code)
- Chronopost integration (https://github.com/scopen-coop/ChronopostWS)
- Roundcube CalDAV integration (https://github.com/scopen-coop/roundcube_caldav)
- Frappe Docker files (https://github.com/scopen-coop/frappe-docker-files)

## Company Structure Notes

**SCOP (Société Coopérative et Participative) Structure:**
- Employee-partners are majority shareholders (at least 51% of capital, 65% voting rights)
- Democratic decision-making (1 person = 1 vote)
- Founded 2020 by four founders: Julien, Marc, Florian, and Nicolas
- Name combines "SCOP" (cooperative structure) + "OPEN" (open-source values)
